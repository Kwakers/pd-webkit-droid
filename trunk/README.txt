WebKitPdDroid

Get your Pure Data patches working on your Android device with an HTML5 + Javascript user interface.

 1. Download WebKitPdDroid.apk or find it in the Android market
 2. Create a user interface for your Pd patch using HTML & Javascript
 3. Place all files in a directory ending with .wpd (e.g. MyPatch.wpd)
 4. Put the .wpd folder on the sdcard in your device
 5. Launch the WebKitPdDroid app

Download test.wpd for a working example.
